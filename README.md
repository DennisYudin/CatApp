# Spring REST app

### Пару слов о проекте
Данное приложение представляет классичеcкий веб-сервис с реализацией таких HTTP запросов как:
- GET
- POST
- PUT
- DELETE

Данное приложение было сделано на основе задания:
https://github.com/wgnet/wg_forge_backend
